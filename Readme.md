# PC Config Builder

## Mô tả
Website xây dựng cấu hình PC với frontend HTML/CSS/JS và backend NodeJS (Express).

## Chức năng
- Chọn linh kiện PC
- Tính tổng tiền
- Lưu cấu hình
- Xuất Excel / Ảnh
- Chia sẻ link cấu hình
- Trang xem cấu hình riêng biệt

## Công nghệ
- Frontend: HTML, CSS, JavaScript
- Backend: NodeJS, Express
- Lưu tạm: RAM (demo)

## Cách chạy
1. Chạy backend:
   node server.js
2. Mở A.html bằng Live Server
